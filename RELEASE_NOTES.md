# Список покупок v0.2.2

Минорное обновление Android-интерфейса.

## Что исправлено

- Поле ввода товара больше не обрезает введенный текст.
- В полях добавления товара убрана плавающая метка внутри рамки, вместо нее используется подсказка внутри пустого поля.
- Расстояние между пунктами списка уменьшено до минимального практичного значения.
- Строки товаров стали плотнее за счет уменьшения внутреннего вертикального отступа.

## Что не менялось

- Серверная логика синхронизации, совместного доступа и приглашений не менялась.
- Инструкции по TrueNAS и Docker остаются актуальными.

## Файлы релиза

- `shopping-list-android-v0.2.2.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v0.2.2.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v0.2.2.zip` - полный архив исходного кода.

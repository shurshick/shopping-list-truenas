# Список покупок v0.2.1

Минорное обновление Android-приложения после релиза v0.2.0.

## Что исправлено

- Исправлено поле ввода товара на главном экране: текст больше не обрезается и не прячется под верхней рамкой поля.
- Сохранен компактный вид блока добавления товара, но высота полей увеличена до безопасного значения для Material `OutlinedTextField`.

## Что не менялось

- Серверная логика совместного доступа, одноразовых приглашений и синхронизации осталась без изменений.
- Инструкции по развертыванию на TrueNAS и Docker остаются актуальными для версии v0.2.1.

## Файлы релиза

- `shopping-list-android-v0.2.1-debug.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v0.2.1.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v0.2.1.zip` - полный архив исходного кода.

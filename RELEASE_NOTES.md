# Список покупок v0.2.0

Первый обычный релиз без статуса предварительной проверки. Версия готова для установки на Android-телефон и разворачивания серверной части на собственном TrueNAS или Docker-сервере.

## Главное

- Android-приложение переведено на более аккуратный интерфейс в стиле Material 3.
- Серверная часть разворачивается через Docker Compose, TrueNAS Custom App, сборку напрямую из GitHub или готовый GHCR-образ.
- Первый запуск сервера проходит через веб-мастер `/setup`.
- При первом запуске обязательно создается пользователь-администратор.
- Списки можно вести совместно между несколькими пользователями.

## Android-приложение

- Главный экран показывает выбранный список покупок без лишней панели со списками.
- Новый список создается кнопкой `+` в верхней панели.
- Меню текущего списка открывается через иконку шестеренки.
- В меню списка доступны участники, переименование, копирование, очистка, удаление, доступ по email и ссылка-приглашение.
- Позиции списка можно отмечать как купленные и удалять через иконку корзины.
- Поля добавления товара стали компактнее.
- Добавлен справочник товаров и автопоиск по части названия.
- Добавлена иконка приложения.

## Совместный доступ

- Список можно открыть зарегистрированному пользователю по email.
- Можно создать ссылку-приглашение.
- Ссылка-приглашение одноразовая: после присоединения пользователя токен больше не используется.
- В меню списка можно посмотреть, кому открыт доступ.
- Если не-владелец удаляет список, список удаляется только у него, а доступ к этому списку отключается.
- Если владелец удаляет список, список удаляется полностью.

## Серверная часть

- Backend работает на FastAPI.
- Данные хранятся в PostgreSQL.
- Настройки сервера хранятся в базе и меняются через `/setup`.
- Публичная конфигурация для приложения доступна через `/server-config`.
- Добавлены API для участников списка, приглашений, очистки списка, копирования, переименования и удаления.

## Файлы релиза

- `shopping-list-android-v0.2.0.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v0.2.0.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v0.2.0.zip` - полный архив исходного кода.

## Быстрый старт

Для TrueNAS 25.04 Custom App используйте `docker-compose.truenas-custom.yml`. После запуска откройте:

```text
http://truenas-ip:8000/setup
```

Создайте администратора, укажите внешний HTTPS-адрес сервера, затем установите APK на Android и войдите в приложение.

Подробные инструкции находятся в README и папке `docs`.

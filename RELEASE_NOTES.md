# Список покупок v0.2.3

Минорное обновление сборки Android APK.

## Что исправлено

- APK теперь подписывается постоянным ключом проекта, который хранится в репозитории.
- Будущие версии APK из GitHub Releases должны устанавливаться поверх предыдущей версии без удаления приложения.
- Убрана проблема, когда Android показывал ошибку "Приложение не установлено, так как оно конфликтует с другим пакетом" из-за разной подписи APK между релизами.

## Важное примечание

Если на телефоне уже установлена версия `v0.2.2` или более ранняя, она могла быть подписана временным ключом GitHub-сборщика. Android не позволяет обновить приложение APK с другой подписью поверх уже установленного приложения.

Поэтому для перехода на `v0.2.3` может понадобиться один раз удалить старую версию и установить `v0.2.3` заново. После этого следующие обновления должны ставиться поверх без удаления.

## Файлы релиза

- `shopping-list-android-v0.2.3.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v0.2.3.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v0.2.3.zip` - полный архив исходного кода.

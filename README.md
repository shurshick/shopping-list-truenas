# Shopping List

Проект состоит из двух частей:

- `backend` - сервер синхронизации для Docker/TrueNAS.
- `android` - Android-приложение на Kotlin.

## Быстрый запуск сервера

1. Скопируйте `.env.example` в `.env`.
2. Замените `POSTGRES_PASSWORD` и `JWT_SECRET` на длинные случайные значения.
3. Запустите:

```powershell
docker compose up -d --build
```

Проверка:

```powershell
curl http://localhost:8000/health
```

## Первичная настройка после запуска

После запуска Docker откройте:

```text
http://truenas-ip:8000/setup
```

Введите `SETUP_TOKEN` из `.env`, внешний HTTPS-адрес сервера и параметры регистрации пользователей.

## Доступ извне

Для доступа из интернета лучше использовать доменное имя и HTTPS через reverse proxy на TrueNAS, например Nginx Proxy Manager, Caddy или Traefik. Публиковать API напрямую без HTTPS не стоит, потому что приложение передает логин, пароль и токен авторизации.

## Синхронизация

Клиент при запуске вызывает `GET /sync` и получает все списки, к которым у пользователя есть доступ. Изменения отправляются на сервер отдельными запросами: создание списка, добавление товара, отметка купленного, удаление.
